# Preferences — Tech Stack Selector

## Štýl

- Slovenčina v markdowne, angličtina pre identifikátory knižníc.
- Tabuľky pre matice. Žiadne dlhé prosaic recenzie.

## Citácie

- Každý externý fakt s URL + dátumom kontroly (`overené 2026-05-08`).
- Pri rozhodovaní necituj subjektívne blogposty ako autoritu.

## Hodnotová škála

- Scoring 0–5 (0 = nevhodné, 3 = OK, 5 = excellent).
- Per kritérium uveď **dôvod** scoringu, nie len číslo.

## Konkrétnosť

- Verzia knižnice (`react@19.x`, `vite@5.x`).
- Licencia presne.
- Bundle size konkrétne (kB gzipped).
