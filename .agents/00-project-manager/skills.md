# Skills — Project Manager

PM používa skills minimálne — väčšina práce je orchestrácia a IO.

| Skill | Kedy |
|---|---|
| `buddy:persona-po` | Pri písaní user-facing summary (release notes pipeline behu). |
| `buddy:persona-scribe` | Pri tvorbe `summary.md` po skončení behu. |
| `buddy:commit` | **Iba ak používateľ explicitne požiada**. PM štandardne necommituje. |

## Custom skills (ak budú)

Žiadne v tejto fáze. Ak v budúcnosti vznikne potreba (napr. validátor
markdown štruktúry), pridaj sem.

## Skills, ktoré PM **nepoužíva**

- `buddy:spec`, `buddy:plan`, `buddy:tasks` — to robia analytickí agenti.
- `buddy:implement` — to je implementačná fáza, mimo scope PM.
- `buddy:persona-architect`, `buddy:persona-security`, atď. — patria
  konkrétnym agentom, nie supervízorovi.
